## 🛠 Project Architecture

This repository contains the full-stack infrastructure for the Cronan Technology® ecosystem:

* **Frontend (Next.js):** A high-fidelity React interface featuring a live launch countdown, interactive "Digital Pixel Board" background, and integrated intake pipelines.
* **Backend (Python Flask):** A secure API handling data ingestion, applicant tracking, and business lead management.
* **Database (SQLite):** A structured data repository for managing our 1099 specialist roster and enterprise-level client leads.
